## 关键发现

### Boss直聘免费账号额度（2026-03-05 实测）
- **推荐页"打招呼"**: 每日 3 次（免费版）
- **搜索页"畅聊卡"**: 每日 1 张
- **搜索页"联系Ta"**: 畅聊卡用完后按钮变成"联系Ta"，点击仍触发购买弹窗
- **热搜牛人**: 需消耗多次畅聊卡（如 3 次），免费用户基本无法联系
- **VIP 高效版** (~10元/天): 60 沟通/天 + 120 查看/天
- → 免费账号每天最多联系 ~4 个候选人，批量招聘必须开 VIP

## 工具与脚本

### scripts/batch_greet.py
推荐页批量打招呼脚本。功能：
- 自动切换职位、按评分排序（3-5年优先、大厂加分）
- 自动点击"打招呼"、翻页加载更多
- 额度耗尽检测（购买弹窗识别）
- 结果保存到 `data/greet_results/`

### 额度限制处理
- 推荐页弹窗关键词: `今日主动沟通数已达上限`、`需付费购买`
- 搜索页弹窗: 职位选择对话框 → 选职位 → 确认用畅聊卡
- 搜索页成功提示: `使用成功 已为您发起沟通并索要简历/微信/电话`
- 弹窗关闭: `.dialog-wrap.active .boss-popup__close`
